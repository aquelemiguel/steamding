F.A.Q.

How does steamding work?

After logging in, Steam provides us your SteamID64, your unique account identifier, and nothing else. We then run it through a simple Steam Web API call to extract some publicly available information like the profile URL, avatar and nickname. If everything's in place, the server then continuously scrapes your community page looking for achievement updates.
It's cutting edge tech, really.

Why does my profile need to be public?

Because of web scraping, which is a fancy word for a robot that looks at a page and parses extracted HTML. It must find which game you're playing and its achievement progress. Given all logic is server-sided, we're watching your profile as any random person would. Still, this may only be a temporary measure as I optimize client-side scraping performance.

Does it work on any browser?

Positive on Chrome, Edge and Steam's browser. Haven't tested on any others but should behave as intended.

Can this get me VAC banned?

It's a website, it doesn't inject anything. Either way, you can skim-through the source code yourself on its GitHub page for any code that oddly feels like I'm trying to steal your karambit.
